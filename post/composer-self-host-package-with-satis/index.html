用 Satis 处理私有资源包

2021-11-26 次阅读

文章目录

    • 安装方式
      • 本地安装
      • Docker 镜像 安装
    • 使用方式
    • satis.json

Satis 是 composer官方推出的一款私有资源包托管工具。项目地址：https://github.com/composer/satis

安装方式

本地安装

1

composer create-project composer/satis:dev-master

Docker 镜像 安装

1
2
3
4
5

docker run --rm --init -it \
  --user $(id -u):$(id -g) \
  --volume $(pwd):/build \
  --volume "${COMPOSER_HOME:-$HOME/.composer}:/composer" \
  composer/satis build <configuration-file> <output-directory>

使用方式

配置 satis.json文件

1
2
3
4
5
6
7
8
9

{
    "name": "satis/repo",
    "homepage": "http://packages.example.org",
    "repositories": [
        // 一行一个写入 gitlab 项目地址
        { "type": "vcs", "url": "git@git-biz.qianxin-inc.cn:seccloud/ssmp-composer-eolinker.git" },
    ],
    "require-all": true
}

运行命令生成包缓存, 将会在packages目录生成索引文件

1

php bin/satis build satis.json packages

生成完成后启动 WEB 服务器

1

php -S localhost:8080 -t ./packages

配置项目composer.json文件

1
2
3
4
5
6

{
    "repositories": [ { "type": "composer", "url": "http://localhost:8080/" } ],
    "require": {
        "company/package": "1.2.0", 
    }
}

更新项目依赖

1

composer update -vvv
